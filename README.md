# Sorumlu Listeleme Alanı
Bu ekranda herhangi bir güncelleme işlemi yapıldığında karşı tarafa mail gidiyor. Örneğin bilgisayar mühendisliğinde x kişisi sorumlu olsun. Ardından y kişisi x kişisinin
yerine sorumlu olarak geldiğinde y kişisinin mail adresi yazıldığında karşı tarafa mail gönderilir.
